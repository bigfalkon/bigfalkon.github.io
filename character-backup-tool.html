This writes the JSON data from the zip back to Firestore in merge mode, using the same document IDs. Image URLs keep working unchanged as long as the original host (imgbb/vgy etc.) is still online.

Advanced: re-upload images (if the host is down / you want fresh URLs)

Re-uploads the local image files inside the zip to ImgBB and updates the URL fields in the documents with the new URLs. Only ImgBB is supported.
