🔥 Firebase Migration Tool

⚠️ Başlamadan önce: Hedef Firebase projesinde Firestore kurallarını geçici olarak aç:
Firebase Console → Firestore → Rules → Publish 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
Migration bittikten sonra kuralları eski haline getirmeyi unutma!
